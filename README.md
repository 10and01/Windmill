Windmill Focus 是一个轻量的专注工具，结合了风车互动、任务管理、番茄钟和本地天气氛围，支持网页与桌面（Electron）双端运行。

## 项目功能

1. 风车交互专注
- 点击风车可以加速旋转，连续点击会进入高速状态。
- 根据当前速度动态生成环境音，增强沉浸感。

2. 任务管理
- 支持新增、完成、删除任务。
- 本地持久化保存任务列表。
- 已完成任务按周自动清理，保持列表简洁。

3. 专注计时器
- 内置倒计时功能，默认 25 分钟。
- 支持输入自定义时长并启动/暂停。
- 可选择是否在风车区域显示计时信息。

4. 自动本地天气氛围
- 通过定位自动获取当前天气（Open-Meteo）。
- 根据天气状态切换界面氛围表现。
- 显示温度、天气描述和风速信息。

5. 背景与音频设置
- 支持自定义背景图片。
- 支持输入背景音乐 URL。
- 支持导入本地自定义音乐文件并立即播放。
- 支持调节音量和音色类型。

6. 桌面浮窗体验（Electron）
- 提供无边框、小尺寸、置顶的专注窗口。
- 支持托盘驻留与快速唤起。

## 开发命令

1. 安装依赖

```bash
npm install
```

2. 启动网页开发模式

```bash
npm run dev
```

3. 启动桌面开发模式

```bash
npm run desktop:dev
```

该命令会自动启动 Vite，并在服务可用后打开 Electron 窗口。

4. 类型检查

```bash
npm run lint
```

## 音乐说明

- 你可以直接粘贴在线音频地址，或者通过设置面板导入本地音频文件。
- 导入的音乐会被转换为可播放资源，并随设置保存。
- 如果想恢复默认音效，可以在设置里点击 Reset。
